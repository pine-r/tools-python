# tools-python
使用python写的小工具  
###环境要求：  
Python3.8.6  
###使用教程：  
1、克隆代码仓  
```
git clone git@github.com:pine-r/tools-python.git
```  
2、安装第三方库  
```
pip install -r requirements.txt  
```
3、运行main.py文件
```
python main.py
```

###功能描述  
1、网络爬虫和信息提取  
2、文件差异比对 FileCompare   
v1.0: &nbsp;&nbsp;基础功能实现，文件**上传**比对  
3、文件字符串替换  
v1.0: &nbsp;&nbsp;实现单个文件特定字符串替换
